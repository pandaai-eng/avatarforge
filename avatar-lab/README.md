# AvatarForge PromptLab

A lightweight, fully local prototype GUI for generating AI avatar systems, prompt packs, script packs, offer copy, and launch assets.

## What it does

- Generates an **avatar identity bible**
- Generates **image prompts** for avatar consistency
- Generates **video / talking avatar prompts**
- Generates **voice + script prompts**
- Generates a **30-day content plan**
- Generates **landing page copy**
- Exports results as **Markdown** or **JSON**

## How to use

Open `index.html` in a browser.

No build step required.

## Suggested stack for production

- Frontend: Next.js / React
- Data: Supabase or Firebase
- Auth: Clerk / Supabase Auth
- Payments: Stripe / Lemon Squeezy / Stan funnel integration
- File storage: Google Drive / S3 / Supabase Storage
- AI APIs: OpenAI, Flux, Runway, Kling, ElevenLabs, HeyGen

## MVP product idea

Sell this as:
- prompt pack product
- avatar brand system toolkit

## Notes

This prototype is intentionally dependency-light so it can be shipped fast and iterated later.
